hypnotoad-server
================

all glory to the hypnotoad

getting started
---------------

```bash
> coffee src/server.coffee
```

then login with

```bash
> curl localhost:8081
```

there is also a cli

```bash
> coffee src/cli.coffee
```

dependencies
------------

 * charm
 * coffeescript
